-------------------------------------------------------------
完成作业的基本功能点：

1. 有简单场景可以工玩家进行游戏

2. 可以使用键盘控制角色行走以及使用武器攻击

3. 场景中至少有两种不同类型的敌人（小怪和Boss），敌人都可发出攻击

4. 可以8方向攻击并杀死敌人

扩展功能点：

1. 敌人具有一定的智能性，在视野范围内发现主角会主动靠近攻击

2. 主角有多种攻击动作，还可以费血放大招

3. 主角血条显示在屏幕左上方，敌人血条显示在敌人头上
--------------------------------------------------------------
游戏运行方式：

进入MyGameDemo文件夹，双击MyGameDemo.exe运行

--------------------------------------------------------------
游戏操作：

基本方向控制：a(向左), w(向上），s(向下)，d（向右）

基本攻击键：j(普通攻击a),k(普通攻击b）

--------------------------------------------------------------
Tips:

1. 当同时按下j和k时，会释放大招(耗费自身一定血量）

2. 释放大招前的方向控制，会影响大招的释放方向，从而可以

   达到8方向攻击（好吧，这个纯粹是为了满足题目要求。。。）

3. 由于素材不足，所以最后的大Boss跟小怪一模一样，只是体积更大，

   血更多，攻击力更高而已。。。

--------------------------------------------------------------
代码说明：

1. src文件夹为源代码，其中src/toolkit.lua为自己封装的一些

   cocos常用函数，方便使用，比如实现了类似C++中的std::bind，方便

   回调成员函数。 src/app文件夹为我们游戏的主要代码

2. 代码以面向对象的方式组织，Init.lua为app的初始化文件，目的是加载

   其他类，其他文件均代表一个同名类。

   Role 角色类，Enemy, Hero,Boss 均继承自Role


3. 所有配置均在Config.lua中，包括精灵的所有动作帧定义，属性等，可以

   方便更换美术素材，以及策划调整人物属性值

4. 代码均包含简单注释，方便查看
